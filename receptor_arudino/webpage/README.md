# Estructura básica de React

Este proyecto contiene la estructura mínima para comenzar una página web con React y JSX utilizando Vite como bundler de desarrollo.

## Scripts

- `npm install` para instalar dependencias.
- `npm run dev` para iniciar el servidor de desarrollo.
- `npm run build` para crear la versión de producción.
- `npm run preview` para previsualizar el build.

## Estructura

```
public/
  index.html
src/
  App.jsx
  main.jsx
  styles.css
  components/
    Header.jsx
```

Puedes añadir más componentes dentro de `src/components` y estilos adicionales en `src/styles.css` según tus necesidades.
